# Sistem İzleme Aracı

Bu Python tabanlı araç, sistem bilgilerini ve performans durumunu komut satırı üzerinden görüntülemenizi sağlar.

## Özellikler
- Sistem Bilgileri
- CPU Kullanımı
- Bellek Durumu
- Disk Bilgileri
- Ağ Bilgileri
- Gerçek Zamanlı CPU Kullanım Grafiği
- Çalışan İşlemler
- Sistem Başlangıç Süresi
- Disk Performansı
